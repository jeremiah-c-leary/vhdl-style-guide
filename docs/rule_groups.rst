Rule Groups
===========

Rule groups allow for easier configuration of similar rules.

The following are a list of the rule groups:

* indent
* case
** case vhdlkeyword
** case label
** case identifier
* blank line above
* blank line below
* prefix
* suffix
* whitespace
* structural
* length
* optional_keyword
* alignment

Configuring a rule group
------------------------

A rule group can be configured using the **group** option
